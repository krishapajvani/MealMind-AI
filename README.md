# 🍽️ MealMind AI

An AI-powered food recommendation web application that suggests personalized meals based on user preferences, dietary needs, and available inputs.

🚀 This project was initially generated using **Firebase Studio AI** and later enhanced with custom features, improved UI, and deployment using modern web technologies.

---

## 🧠 About the Project

MealMind AI helps users discover meals tailored to their needs using artificial intelligence. Users can input preferences such as diet type, ingredients, or goals, and the system generates smart meal recommendations.

This project demonstrates the integration of AI with modern web development frameworks to build a real-world application.

---

## ✨ Features

* 🤖 AI-powered meal recommendations
* 🧾 Input-based personalized suggestions
* ❤️ Save and manage favorite meals
* 👤 User profile interface
* 📱 Fully responsive design
* ⚡ Fast and optimized performance

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router), React
* **Styling:** Tailwind CSS
* **AI Integration:** Firebase Studio AI / Genkit
* **Backend Logic:** API-based AI processing
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/krishapajvani/MealMind-AI.git
```

### 2. Navigate to project directory

```bash
cd MealMind-AI
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run development server

```bash
npm run dev
```

### 5. Open in browser

```
http://localhost:3000
```

## 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```env
NEXT_PUBLIC_API_KEY=your_api_key_here
```

> ⚠️ Never expose sensitive API keys publicly.
