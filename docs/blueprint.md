# **App Name**: MealMind AI

## Core Features:

- AI Meal Recommendation Engine: Users provide their dietary preferences, available ingredients, current mood, or occasion, and the AI will use its reasoning to tool personalized meal recommendations, offering diverse and suitable recipe suggestions.
- User Profile & Preferences: Allow users to input and manage their dietary restrictions (e.g., vegetarian, gluten-free), cuisine preferences, allergies, and ingredient dislikes to refine recommendations.
- Detailed Recipe Display: Present recommended dishes with comprehensive information, including ingredient lists, step-by-step cooking instructions, and estimated preparation time.
- Save & Favorite Meals: Enable users to save their favorite meal recommendations for quick access and future reference, implying persistent storage.
- Recommendation Feedback System: Provide a mechanism for users to rate or provide feedback on recommendations to continuously improve the AI's future suggestions based on their tastes.

## Style Guidelines:

- Primary color: A sophisticated, muted forest green (#476647), evoking natural ingredients and healthy eating. It provides a grounding, earthy feel for key interactive elements.
- Background color: A very subtle, desaturated light green (#F1F6F1), offering a clean and fresh canvas that hints at nature without overpowering the content.
- Accent color: A warm, inviting golden ochre (#CC993D), chosen for its ability to create a harmonious contrast and highlight important actions or visual cues, reminiscent of rich flavors.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly tech-forward feel. Body font: 'Inter' (sans-serif) for excellent readability and a neutral, professional appearance, complementing the headlines.
- Utilize simple, clean line-art icons related to food, cooking, and user interface actions, ensuring clarity and a modern aesthetic.
- Implement a clean, spacious card-based layout for displaying meal recommendations, ensuring readability and visual hierarchy on various screen sizes.
- Incorporate subtle hover effects on actionable elements and a smooth, gentle transition when new meal recommendations are loaded or filtered, enhancing user engagement without distraction.
